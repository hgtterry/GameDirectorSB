Change Log
 

EquitySB 23/10/22

Added:- Collectables Entity.
This Entity adds an object to the scene but when you come into contact it they will disappear and play a sound if required.
It used mainly for collectables like gems weapons or other collectable items this additions is part 1 of 2 in the next release we will be adding Panels that will display Collected items and Scores or just plain Text.

Updates- Mesh Viewer.
Items selected in the Mesh Viewer can now be rotated and Zoomed useing the Mouse. Still needs more work.

General.
More Help Files. Videos will be added as we go along.
Bug fixes and Stability Issues.


Thanks
HGTO Software:- Terry and Hazel Flanigan.

EquitySB 16/10/22

Added:- Teleport Entity.
This adds a Entity Trigger Allowing the Player to be Moved/Teleported to different parts of the scene.

Added:- Select Objects.
Clicking the select Objects button allows for quick Selection of items especialy usefull with a scene with a lot of items in.

General.
Started on Help Files. Videos will be added as we go along.
Bug fixes and Stability Issues.


Thanks
HGTO Software:- Terry and Hazel Flanigan.

EquitySB 09/10/22

Added Tri-mesh Physics Shape.
This adds a Physics bounding box that is the exactly the same shape as the mesh useful for for Tunnels Steps Bridges and complex shapes.

Added :- Move Entity.
This Entity adds the ability to move another static object in the scene and is useful for things like Doors Secrete openings. The Move Entity can be scaled and moved within the scene as a Trigger even in a different are.


Thanks
