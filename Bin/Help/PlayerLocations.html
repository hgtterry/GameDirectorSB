Player Locations
 
The player locations Dialog contains controls that enable you to save and select locations.

The First locations called Start_Location can not be deleted or renamed and is created for you when
the project was first created and is the main start position for the player and can be edited only by click on the player in the fileview.



Clicking this button will move the player to the selected location you selected in the List Box.This will also set Physics on and change you to first player mode you are the Player.


Edit allows you to change the name of the selected location.

Delete will delete the selected location note if you have used the deleted location in a Teleporter Entity the Teleporter will still work as it copies the details into the Entity.


In general you will move your player round as normal in first player mode then click Add New Player Location and this will save a new location where you currently are.

But sometimes you cant move your player say to a new area and get through the walls so to make this possible select FreeCam then move the camera into the area then click the Player->Camera button this will move your player through the walls to the camera.Once in side the new area you can move round as normal and when happy Click Add New Player Location as normal.






Click Add New Player Location will bring up a Dialog asking for a name click ok and the location where you are will be saved .
