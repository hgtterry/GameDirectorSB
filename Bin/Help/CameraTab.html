Camera Tab
 







Clicking the First button will attach the main camera to the player in the first view prospective and also set the Physics to on.
The button will toggle Between white and green to indicate the state.
Not Available at the moment.



The Free Cam Button is the default selected when your project loads this gives you the ability to go any where within the scene through walls and any solid structures and causes no triggers and has no Physics attributes.



The Select button allows you to Select an Object within the Scene. Some times this can be useful when there are lots of objects. A round Cross hair will appear in the centre of the screen use use this and place over an object then press space bar a conformation Dialog will appear and ask you do you wish to edit the object.

Brings up a Dialog with various speed settings theses speed settings only affect the free cam speeds and not the Player speeds.
