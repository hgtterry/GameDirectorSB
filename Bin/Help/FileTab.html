File Tab
 



The Quick button allows you to load a project quickly with out any conformation dialogs and not using the file explorer. When you first use GameDirectorSB for the first time the default project will be loaded the one we use to test the application.

In general any new or existing project you load then save will be set to the Quick Load button saving you time looking for the project.
