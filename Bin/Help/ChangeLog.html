GameDirectorSB 15/01/23

Added:- StartScreen\Front Dialog.

Will bring up a Dialog on your Built games with Start Game and Quit. Once the the game as been started and you press Escape Key the Dialog will return with Resume, Restart, and Quit. This is also available in Game Mode so you can test.

We have just implemented this so in the basic stages but over the next couple of releases we plan to make it fully customizable.

Note:-  the Front Dialog is set on by default just uncheck Front Dialog option when you test in Game Mode or Build your Scene\Game


Updated:- EquityME in Tools Folder.

Updated:- World Editor in Tools Folder.

GameDirectorSB 07/12/22

Updated:- Build Game
This will Build a Stand alone Scene\Game.

  • Added Options for displaying FPS and Also Full screen or Desktop Mode for your stand alone Scene\Game.

Updated:- Counters.
Counters:-  Will be used for logic decisions such as needing a certain amount of collectables before opening a gate as an example.

Used for Messages,Collectables, and Move Entities. More will be added as we go along.


Updated:- Locations
The Locations Tab is where Player and the Camera locations can be saved or selected and moved this can be useful  for complex scenes where you might be editing or testing part of a scene and need a quick way to get there.
Locations saved can also be used in Entities like the Teleporter Entity.

  • Bug Fixes

Added:-Dimensions Locks.

when you have finished moving rotating and scaling an object they can be locked when you are happy.

Added:-Environments Entities.

You can now create Environment Entities theses will change the environment settings such as Main colour Fog colour and the back ground music.

General.
Bug fixes and Stability Issues.

More Help Files. Videos will be added as we go along.
