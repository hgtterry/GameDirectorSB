Export Mesh
Exports the mesh with its vertices normals and texture cords

Joints/Bones
If the model has a joint bone structure (Skeleton) this will be exported if the model has no joints the tick box will be grayed.

Export Motions
Exports Motions must have a Joiint/Bone structure if not will be grayed.

Dummy Material File
Creates a Dummy Material file.


Please Note these option affect the Exported Model only so if you have a Skeleton and motions in the model in Equity they will be preserved.
